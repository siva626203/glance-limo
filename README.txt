Glance Limousine production-ready split files

Files:
- index.html
- styles.css
- app.js

How to use:
1. Keep all 3 files in the same folder.
2. Open index.html in a browser for preview.
3. Upload all 3 files to your server in the same directory.
4. For live checkout, your developer should replace the current Stripe-ready demo buttons in app.js with real Stripe Checkout or Payment Links.

Notes:
- The current setup is front-end only.
- Payment buttons are structured and ready for real integration.
- WhatsApp booking actions are already wired on the front end.
